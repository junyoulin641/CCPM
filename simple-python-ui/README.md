# Simple Python UI

A minimal PyQt6 desktop application demonstrating basic UI components and event handling.

## Requirements

- Python 3.8 or higher
- PyQt6

## Installation

Install dependencies using pip:

```bash
pip install -r requirements.txt
```

## Running the Application

Run instructions will be added when main.py is created.
